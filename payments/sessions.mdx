# Create a new Session

- **Endpoint URL:** `/checkout/sessions`
- **HTTP Method:** `POST`

### Description

Initialises a new checkout session.

### Parameters

List and describe all the parameters required for the endpoint.

#### Request Parameters

- **Customer*:**
  - **Name:** `customer`
  - **Type:** `string`
  - **Description:** An indentify you use to reperesent a customer within your system.

- **Product Id*:**
  - **Name:** `product_id`
  - **Type:** `string`
  - **Description:** The Product id of the product the checkout session is created for.

- **Callback URL (Optional):**
  - **Name:** `callback_url`
  - **Type:** `string`
  - **Description:** `Url to redirect your customers to after payment`

### Status Codes

- **200 OK:** Successful request.
- **400 Bad Request:** Invalid parameters or missing required parameters.
- **404 Not Found:** Resource not found.
- **500 Internal Server Error:** Something went wrong on the server.

